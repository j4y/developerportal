Affdex SDK for iOS Release 3 Documentation

Welcome to the Affdex SDK for iOS! With this SDK, your iOS app will be able to detect faces and their facial expressions using the built-in camera, or via a file on your iOS device.

Hardware Requirements

  • Any iPad, iPhone, or iPod Touch running iOS 8 or newer.

Development Requirements

  • Xcode 7 with the iOS 8 SDK

Getting Started
To get started, visit https://github.com/Affectiva/ios-sdk-samples and pull down the source for our demo apps. Ensure that the Code Signing settings for the various projects are setup correctly.
